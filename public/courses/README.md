# Zdjęcia kursów

Wrzuć tutaj wygenerowane zdjęcia do oferty kursów.

**Sugerowane nazwy plików** (żeby dopasować do `image_url` w `src/data/courses.ts`):

- `lekcje-indywidualne.jpg` (lub .png, .webp)
- `lekcje-grupowe.jpg`
- `powtorka-maturalna.jpg`

Po dodaniu plików ustaw w `src/data/courses.ts` np.:

```ts
image_url: "/courses/lekcje-indywidualne.jpg",
```

Format: poziomy, min. 800×500 px (np. 16:10 lub 4:3).
